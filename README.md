# Сайт ТОВ «РЕЗЕРВ ЕНЕРДЖИ»

Корпоративний сайт компанії з оптової торгівлі електроенергією.
Побудовано на **чистому HTML / CSS / JavaScript** — без фреймворків, без збірки.

---

## 📁 Структура проекту

```
reserv-energy/
├── index.html          — Головна сторінка
├── about.html          — Про нас
├── services.html       — Трейдерська діяльність
├── documents.html      — Нормативні документи
├── contact.html        — Контакти (з формою)
├── css/
│   └── style.css       — Уся стилізація (дизайн-система)
├── js/
│   └── main.js         — Мобільне меню, анімації, форма
└── assets/             — (для майбутніх зображень)
```

---

## 🚀 Як запустити

**Варіант 1. Відкрити напряму.** Достатньо відкрити `index.html` у браузері
(подвійний клік по файлу) — сайт працює без сервера.

**Варіант 2. Через локальний сервер** (рекомендовано для коректної роботи шрифтів):

```bash
# Python 3
python3 -m http.server 8000

# Node.js
npx serve .
```

Потім відкрити http://localhost:8000

## 🎨 Дизайн-система

- **Кольори**: темне тло `#0a0e14` + електричний синій `#0b5fff` + енергетичний жовтий `#ffd400`
- **Шрифти**: Space Grotesk (заголовки) + Manrope (текст) — через Google Fonts
- **Іконки**: всі SVG вшиті inline
- **Анімації**: плавна поява блоків при скролі, анімована енергетична хвиля в hero
- **Адаптивність**: breakpoints 1024 / 768 / 480 px

## ✏️ Що змінити перед публікацією

1. **Контакти** — в прикладі використано `+380 44 360 1234` та `info@reservenergy.ua` як placeholder. Замініть на реальні у всіх `.html` файлах (можна через Find & Replace).
2. **Посилання на законодавчі акти** в `documents.html` — зараз placeholder'и на zakon.rada.gov.ua та nerc.gov.ua. Замініть на фактичні URL.
3. **Форма в contact.html** — зараз це JS-симуляція надсилання. Для реальної роботи підключіть бекенд (PHP, Formspree, Google Forms, тощо) або замініть `event.preventDefault()` на нормальний submit з `action="..."`.
4. **Соцмережі** — якщо потрібні, додайте лінки у футер.
5. **Favicon** — зараз SVG-блискавка. За потреби замініть на ваш логотип.
